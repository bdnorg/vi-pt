Cloud sync (Google Sheet)

One-time setup: create a Google Sheet → Extensions → Apps Script → paste the script from the setup guide → Deploy → New deployment → Web app, execute as Me, access Anyone → copy the URL here. Your exercises live in an editable Exercises tab — add or change rows in the sheet, then ⬇ Load from cloud — and every finished session is appended to a History tab.
